Sobre el Sistema de Inventario

Este Sistema de Inventario está diseñado para ayudarte a gestionar tus habitaciones y cajas de manera eficiente. Puedes crear habitaciones, añadir cajas y organizar tus artículos con facilidad. El sistema está construido utilizando tecnologías web modernas como HTML, CSS (SASS) y Bootstrap.

Ya sea que estés gestionando un pequeño espacio de almacenamiento o un gran almacén, esta herramienta proporciona una forma simple e intuitiva de llevar el control de tu inventario.
